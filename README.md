# Hynfratech


## Installation

### 1. Clone the repository:

   ```bash
   git clone https://github.com/antonymwangig/hynfratech.git
   cd hynfratech
   ```
### 2. Install dependencies:
``` 
  npm install
```

### 2. Development
To start the development server with hot-reload functionality:


``` 
  npm run dev
```
